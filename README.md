## 작성 규칙

- 언어나 기술명으로 폴더를 만든다. (root에 문서를 만들지 않는다.)
- 파일명은 영어
- 대략적인 기술만을 목적으로 한다. 상세적인 부분은 [블로그](https://kdm-korea.github.io/blog/)를 사용한다.
