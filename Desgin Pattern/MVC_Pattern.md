# Design Pattern

## Definition

**Design Pattern**이란 소프트웨어 공학에서 프로그램을 설계하는 기법이다. 프로그램의 각기 ```pattern```에 따라 효율성이 향상될 수가 있다.

## MVC Pattern

**MVC**는 ***Model***, ***Control***, ***View*** 의 약자이다. MVC에 대한 설명과 간단한 프로그램을 예시로 들겠다.

### Model

Model은 program에서 수행되는 로직으로 이루어져 있으며

```java
Class Model{

    private int name(){

    }
}
```
